## ClientSideServer

Aplicacion desarrollada con la libreria vuejs, diseÃ±ada para consumir un api sobre consultas de informacion de servidores asociados a un dominio

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

>La apicacion se desplego con los servios hosting de zeit.co, usando su herramienta CLI (now), es una aplicacion responsive, por tanto puede ser accedida desde un telefono movil.
